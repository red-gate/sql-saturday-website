SQL Saturday


This is the homepage for SQL Saturday events. Note that this it an initial site, and will change over time.

SQL Saturday is a collection of independent, local events that are run by community members. Each is typically a one day event, serving the local database professionals in a community. If you are interested in running your own event, you can get information on new events and submit a request to be listed on this page and hosted on this site. You can see a list of future events below, as well as an archive of past events.

This is the fourth evolution of SQL Saturdays, to be managed by an independent foundation established by Redgate Software following their purchase of the assets of PASS. You can read more about this on the About SQL Saturday page.
